Prompt Design That Works

❌ Bad Prompt:

"Build a notification system"

✅ Good Prompt:

Goal: Implement native push notifications Constraints: - Direct APNs HTTP/2 (no Firebase) - Node.js only - 3 retry attempts - Environment variables for keys Deliverables: 1. src/services/apnsService.js 2. src/services/fcmService.js 3. Error handling 4. Test commands

The Iterative Process

1. Human: Define constraints ↓ 2. AI: Generate code ↓ 3. Human: Test immediately ↓ 4. Human: Request fixes ↓ 5. AI: Refine ↓ 6. Repeat until production-ready

Key Lessons Learned

What Worked:

  • Start with vision → Press release first
  • Constrain scope → Stage 1 only
  • Atomic prompts → One feature at a time
  • Test everything → Immediately

What Failed:

  • Long prompts → AI gets confused
  • Assuming context → Must restate constraints
  • Skipping tests → Broken code that looks right
