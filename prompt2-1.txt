Prompt 2.1: React Native SDK (Core & Push)
Goal: Create minimal RN SDK for native push notifications.
Requirements:

Zero-config initialization
Plain JavaScript (no TypeScript) to match backend
Support RN 0.70+
Direct native module integration (no Firebase)
Handle all app states (foreground/background/quit)
